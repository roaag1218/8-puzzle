# 8-puzzle
This project implements a solver for the 8-Puzzle problem using three search algorithms: IDS (Iterative Deepening Search), UCS (Uniform Cost Search), and A* Search with Manhattan distance heuristic.

## Project Structure
8-puzzle/
├─ src/
│ ├─ IDS.py
│ ├─ ucs_8puzzle.py
│ ├─ 8-puzzle_using_A_star.py

## Requirements
- Python 3.7+

## How to Run
Open a terminal in the project root folder and run **any algorithm file directly**:

```bash
python src/IDS.py
python src/ucs_8puzzle.py
python src/8-puzzle_using_A_star.py
